Movie data Analysis Project

## Overview
This project contains an Excel dataset that organizes and analyzes films by attributes such as title, release year, genre, ratings, box office performance, and critical reviews. It is designed to help users explore trends in cinema and compare movies across different dimensions.

## Features
- Structured dataset in Excel format
- Sorting and filtering for quick insights
- Comparative analysis of commercial success vs. critical reception
- Useful for students, researchers, and film enthusiasts

## Usage
1. Download the `movie analysis.xlsx` file.
2. Open in Excel or any compatible spreadsheet tool.
3. Use filters and charts to explore trends and patterns.
